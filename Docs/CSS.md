# *How To Use Custom CSS for The Message?*

<br>

## *Use Custom Font in Msg.js :-*
```CSS
.msg-text{
    font-family: Font-Style;
}
```

<br>

## *Use Custom Font-Size in Msg.js :-*
```CSS
.msg-text{
    font-size: 25px;
}
```

<br>

## *Use Custom Text Colour :-*
```CSS
.msg-text{
    color: green;
}
```

<br>

## *Use Custom Text Background Colour :-*
```CSS
.msg-text{
    background-color: yellow;
}
```
